# Movie Recommender System

A simple Movie Recommender Web App built using Python and Streamlit that recommends movies based on similarity.

## Features

* Select a movie
* Get recommended movies
* Interactive web interface

## Tech Stack

* Python
* Pandas
* Scikit-learn
* Streamlit

## Run the Project

Install dependencies

pip install -r requirements.txt

Run the app

streamlit run app.py
